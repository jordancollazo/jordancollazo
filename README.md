# Jordan Collazo

Operations and Finance Analyst in the Florida Army National Guard. Self-taught builder of production AI tooling.

I build agentic systems with Claude Code: multi-agent pipelines, browser automation, and self-improving prompt loops. Most of what I make starts as a real problem in my own work and turns into a tool other people can use.

### What I build

- **[Carousel Generator](https://github.com/jordancollazo/carousel-generator)** — A four-stage pipeline (research, 10-agent design consensus, HTML generation, Chrome DevTools export) that turns any topic into a fully designed Instagram carousel and exports it as 1080x1350 PNGs. Every stat on a slide is grounded in a fetched source, not invented.
- **[Autoresearch](https://github.com/jordancollazo/autoresearch)** — Autonomous self-improvement loops for prompts, code, and copy. Define one metric, one lever, and binary eval criteria; the agent mutates, scores, keeps winners, reverts losers, and logs every run as a git commit.
- **[Airbnb Pricing Engine](https://github.com/jordancollazo/airbnb-pricing-engine)** — Dynamic nightly-rate modeling for short-term rentals.

### How I work

- Agentic over single-shot. I design multi-stage pipelines and multi-agent fan-out with real aggregation, not one big prompt.
- Grounded. Generative output gets tied to fetched sources so it does not hallucinate numbers.
- Measured. When something can be scored, I put it in a loop and optimize against the metric.

### Stack

Python, TypeScript, Next.js, SQLite and Postgres, Chrome DevTools Protocol, the Anthropic SDK, and Claude Code as the primary build environment.

Background in operations and finance analysis (S1/S3/S4 reporting, data standardization across sites). I came to software from the problem side, not the credential side, and I finish what I build.
